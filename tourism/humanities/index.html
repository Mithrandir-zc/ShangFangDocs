# 上方山人文景观资料

# 简介

上方山集儒、道、佛文化于一域，自古是历史文化名山。史料记载佛教历史有1960 余年(东汉光武10 年)，有寺、庵、碑、塔、摩崖石刻、名人诗词题字等200 余处。目前完好的寺庵有兜率寺、华严洞、藏经阁等26 处，有寺庵遗址药师殿、普贤殿等18 处，有佛殿壁画22 幅、石碑石刻20 处、石经幢53 通、古塔57 座、古井11 眼、古桥5 座、古云梯1 座。历史上珍藏的两套大藏经、明刻四十二章经、当代华严祖师肉身舍利，称为上方山“佛界三珍”。 位于房山区的上方山国家森林公园，集聚槐、柏、松、檀等古树18种、2万余株；建有寺、庵、碑、塔、摩崖石刻名人诗词题字等200余处；富存着峰、谷、洞、坑、崖、崮等地质遗迹，可谓是资源丰富。这就是自古被称作历史文化名山的上方山。

# 历史

  • 东汉年间，六聘山（现上方山）开始建立寺庙。
  • 隋未唐初，六聘山形成以下院、中院和兜率寺三座寺院为主的寺庵群，总称“六聘山天开寺”。
  • 辽、金时期，六聘山因兜率寺又称上方寺，渐改称上方山，上方山寺庙达到鼎盛时期，形成以兜率寺为中心的七十二庵群。
  • 金朝末年，上方山诸寺毁于战火。
  • 元朝年间，上方山诸寺部分复建。
  • 明嘉靖年间，上方山兜率寺重修。
  • 明万历年间，上方山兜率寺重修。
  • 1980年，云水洞扩展洞口，在洞内修筑水泥游径，并安装照明彩灯，方便游览。
