# 上方山寺庙群

# 寺庙群简介

天下名山僧占尽。在上方山上寺院分布比较广,形成了一座完整体系的寺院群。天开寺、中院寺、上方寺,这三座寺庙同属一个山门。 天开寺创建于东汉时期，天开寺的方丈或者上方居住的寺庙叫上方寺，上方寺坐落的山由此得名叫上方山。北京第一名寺的说法就不足为奇了。 佛教传入中国不久，就传入了上方山地区。《天开中院碑记》和《重修天开寺碑记》，同时记载着，上方山周边的天开寺建于汉,经晋、南北朝、盛于辽、废于金，这个遥远的历史,勾勒出了上方山的佛教文化脉络。

# 高僧结庵

而早在1400多年前的东魏时期，上方山就有高僧结庵而居。一千多年来，上方山更替了一幕幕繁华，成为北京西南著名的佛教名山。 传说，开山祖师华严老祖云游来到上方山，被上方山的奇景迷住了，打算建寺,但山有毒龙,便与独毒斗法,毒龙派雷公电母,一个霹雷将上方山东南劈出一道裂缝,便是现在的云梯,最后毒龙败了,败后将此处所有水泄去天开龙潭安营扎寨。华严祖师追到天开要水,斗法，毒龙又败，只能交出潭水。毒龙说，给水可以，但只给一斗笠清水。华严祖师用斗笠取了水,放在了上方山东北一个叫一斗泉的地方,上方山便有水了,所以此处称为一斗泉,华严祖师在此处建立斗泉庵。

# 高僧百咏南禅师

北齐范阳人写的《百咏南禅师塔记》中记载，东魏武帝天平二年（535年）,禅宗高僧百咏南禅师在此开山建寺。这里说的开山建寺，并不是上方山建寺之始。他在西峰骆驼峰下选了一块地，建了一个庵。禅宗由上方山作为起点传播到幽州各个寺院，比华严宗传戒台寺早一百六十余年。 禅宗传入中国不久，百咏南禅师就把禅宗传到了上方山，这在上方山志中有明确记载，由此可以说上方山是北京地区最早确定宗派寺院的地方。

# 姚广孝

姚广孝，明朝政治家、佛学家，文学家。朱棣由一个受封的燕王变成大明帝国的皇帝，而帮助他完成这场蜕变的重要谋臣就是姚广孝。永乐称帝后，他想到的不是争功邀宠，而是走回佛堂，完成他的修行事业。于是永乐帝封他做最大的僧官,并将房山大韩继的古代的香光寺封给他作为别墅。这个时候姚广孝看中了南华严林木幽静,挨着上方山,所以他就选择重修此寺作隐居，南华严成为明代众多华严寺中最早恢复香火的一座寺院。

# 华严圣地

华严宗最早传到北京地区是潭柘寺。在房山，华严宗最早的记载是辽代，传颂华严宗的是哪座寺院呢？大房山主峰有座灵峰寺，建于唐代，传颂华严宗；大韩继的香光寺也是颂持华严经的。金代以后，除禅宗外，华严宗在上方山一带传播开来，有南华严、北华严、下华严。到元代，华严宗形成气势，明代华严宗纷纷恢复，并得以兴旺。 明成化十三年,北华严得以恢复,成化二十二年华严弟子把龙王庙修复扩建形成一座华严宗寺院,由此在明代形成了一个围绕上方山形成的完整华严宗体系。无论在西山还是在北京,华严宗这样一脉相承的完整体系是绝无仅有的,所以称上方山是华严宗圣地,是毫无疑问。

# 为何上方山佛教寺院群体堪称全北京之最？

上方山地处北京最西南,位于房山域西南上方山。是进入北京的门户,风景秀美幽静,有九洞十二峰七十二庵古文化圈。它全盛时期的庵不止七十二,据记载有一百二十所,四百公顷的上方山分布着一百二十座寺院,密度非常大。所以被称为北京之最的佛教寺院群体当之无愧。自东汉肇建寺庙，历代相续，或修旧，或扩建，或新辟，辽、金两代达到鼎盛，形成以兜率寺为中心的茅庵寺庙群。兜率寺创建于隋末唐初，位居群山中峰，依山而建，规模为上方寺群之冠，又名上方寺。明万历间太监冯保重修，清代又多次修葺。规模宏大，有正殿、地藏殿、文殊殿、藏经楼等主要殿宇。正殿前有上方山——兜率寺明碑3块，记述该寺兴衰。殿内后檐墙上镶嵌经版15块，镌刻42章佛经。寺西南南塔院，保存辽、金、元、明、清五代许多墓塔，以辽道宗大安五年（1089年）建“六聘山天开寺忏悔上人坟塔记”历史最久，距今已有900余年历史。墓志中记载该僧“所度黑白僧众二十余万，住持本山三十年”。以清乾隆二十四年（公元1759年）建“圆寂伏魔堂第一代浩如泉和尚觉灵塔”最为美观。上方山诸寺大部已废，现仅存十六庵，能开放游览者仅八庵，为接待庵、云梯庵、望海庵、大慈观音庵、胜泉庵等。1984年5 月公布为北京市第三批重点文物保护单位。
