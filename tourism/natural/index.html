# 上方山自然景观资料

# 简介

中国北京房山世界地质公园，地跨北京市房山区和河北省保定市涞水县、涞源县、地理坐标：东经114°36′48″～116°08′16″，北纬39°09′57″～39°43′08″。公园东西长130.80km，南北宽75.09 km，总面积953.95 km2。划分为8个园区：周口店园区、石花洞园区、十渡园区、百花山－百草畔园区、上方山－云居寺园区、圣莲山园区、野三坡园区和白石山园区。

# 地质资源

上方山是中国北方喀斯特分布典型地区，是2012 年公布的中国北方喀斯特地貌遗产的重要组成部分。早在1902 年，法国地质学家普意雅在这里测绘了云水洞，30 年代中国地质学家尹赞勋写了《北京云水洞》一书。这一地区主要是石门背斜形成的岩溶地貌，公园内的峰、谷、洞、坑、崖、崮等地质遗迹资源类型丰富，天柱峰是典型的岩溶孤峰，龙虎峪是典型的峡谷，天坑是中国北方首次发现的地质遗迹。最为典型的是地质变迁形成的云水洞，是目前世界相对海拔高度最高的水平溶洞，属新生代第四纪地质遗迹，石钟乳、石笋、石幔、石柱、石花等沉积物类型齐全。第二厅的石笋，高38 米，在世界溶洞景观中堪称奇观。1980 年在洞中发现古生物化石，经当时中科院贾兰坡院士鉴定为与周口店北京猿人遗址同属一个地质时期的生物化石，具有很高的科研价值。

# 森林资源

它有着适于林木生长的良好生态环境。这里的森林覆盖率高达95%以上。植被丰富，以侧柏林、辽东栎林、低山沟谷杂木林等类型为主。属北温带低海拔落叶阔叶林和针叶阔叶混交林地带，古柏、古栎、青檀群落常伴其间。 　　资料记载，这里有植物110科、319属，652种，其中包括小叶中国蕨、黄檗、绶草等10种国家重点保护植物；槭叶铁线莲、拐枣、野生流苏等北京市重点保护植物21种；房山翠雀、房山紫堇、上方苔草等独有、特有植物23种；以及槐、柏、松、檀等古树18种、2万余株，其中能独领风骚的当属汉柏、辽松、宋国槐、金银杏、元菩提、明古檀等树王。不仅如此，在652种植物里，光药用植物就有537种。
　　总面积达3.29平方公里的上方山因负氧离子浓度是一般空气的8倍，因此又有‘天然氧吧’之美誉。 除了植物资源丰富外，上方山还有陆生野生动物149种，其中两栖动物2种、爬行动物7种、兽类35种、鸟类105种，国家级重点保护动物18种，北京市级重点保护动物80种。 本区在动物界还有独有的硬肢马陆和大足鼠耳蝠，在植物界有5种以本区附近百花山命名的特殊植物种属。
